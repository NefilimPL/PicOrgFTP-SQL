Instalacja Chrome / Edge
  1. Pobierz ZIP i rozpakuj go do stalego katalogu.
  2. Otworz chrome://extensions albo edge://extensions.
  3. Wlacz Tryb dewelopera i wybierz Zaladuj rozpakowane.
  4. Wskaz rozpakowany folder picorgftp-sql-browser-extension.
  5. Na stronie produktu kliknij ikone dodatku, zeskanuj strone i wyslij zdjecia.
  6. W tym oknie kliknij Odbierz z rozszerzenia.
