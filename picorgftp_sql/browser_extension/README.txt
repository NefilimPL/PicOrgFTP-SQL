PicOrgFTP-SQL Image Import

Instalacja Chrome / Edge:

1. Rozpakuj plik ZIP do stalego katalogu, np. Pulpit\PicOrgFTP-SQL Extension.
2. Chrome: otworz chrome://extensions.
   Edge: otworz edge://extensions.
3. Wlacz Tryb dewelopera.
4. Kliknij Zaladuj rozpakowane.
5. Wybierz rozpakowany katalog picorgftp-sql-browser-extension.
6. Otworz strone produktu, kliknij ikone rozszerzenia i wybierz Skanuj strone.
7. Po wyslaniu zdjec w panelu PicOrg kliknij Odbierz z rozszerzenia.

Uwaga:
Paczka pobrana z panelu zawiera adres panelu i token wazny dla biezacego uzytkownika.
Nie udostepniaj tej paczki osobom, ktore nie powinny wysylac zdjec do panelu.
